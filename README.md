

# Anthropic Academy Practice Repo

- Explore and test concepts from the Claude API.
- Serve as a sandbox for learning and experimentation.

---

## 📂 Current Contents
- Files and exercises from **Building with the Claude API** course.
- Example implementations and practice snippets.
- Notes and experiments related to API usage.

---

## 🚀 Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/alharet7/Anthropic.git
   ```
2. Navigate into the repo:
   ```bash
   cd Anthropic
   ```
3. Review course files and run experiments as needed after adding your Anthropic API Key and including it in an .env file.
